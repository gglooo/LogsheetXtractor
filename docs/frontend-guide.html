Frontend Guide

User-oriented walkthrough of the main UI flows in LogsheetXtractor with screenshot placeholders.

Before You Start

  • Application is running and backend API is reachable.
  • OCR credentials are available (uploaded or configured).
  • You have a template PDF and sample logsheet scans.

Workflow Overview

  1. Create or open a template.
  2. Define ROIs and validation conditions.
  3. Upload logsheets and start processing.
  4. Review warnings and proofread extracted values.
  5. Export final data.

Troubleshooting

  • If processing fails, check OCR credentials and template association.
  • If warnings persist, verify ROI validation rules and corrected value format.
  • If alignment is incorrect, re-run manual alignment and re-process.
